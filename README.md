# Clinic Management System

诊所管理系统

## 项目结构

```
clinic-management-system/
├── frontend/          # 前端 - Vue 3 + TypeScript + Vite
└── backend/           # 后端 - Spring Boot + MyBatis-Plus
    ├── clinic-common/  # 公共模块
    ├── clinic-pojo/    # 实体类模块
    └── clinic-server/  # 服务端模块
```

## 技术栈

### 前端
- Vue 3 + TypeScript
- Vite
- Element Plus / Ant Design Vue

### 后端
- Spring Boot
- MyBatis-Plus
- MySQL
- Redis
